# 🤖 Twitter Bot Detector

An AI-powered tool to detect fake accounts and bots on social media, built to protect democratic discourse from disinformation.

## 🎯 Purpose
This tool was developed as part of an initiative for the **World Forum for Democracy 2026** to combat bot-driven disinformation campaigns that threaten free and fair democratic participation.

## ⚡ How It Works
- Uses **Random Forest** machine learning algorithm
- Trained on 37,000+ real Twitter accounts
- Achieves **88% accuracy** in detecting bots
- Analyzes: followers, friends, activity patterns, account age

## 🚀 Run Locally
```bash
pip install flask scikit-learn pandas joblib
python app.py
```
Then open: http://127.0.0.1:5000
